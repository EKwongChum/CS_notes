# ADB Commands

| 序号 | 命令                                          | 含义                             | 备注 |
| ---- | --------------------------------------------- | -------------------------------- | ---- |
| 1    | adb shell ping ekwong.cn                      | 检测设备的网络能否连上对应的域名 |      |
| 2    | adb pull sdcart/path/to/file.txt pc/path/file | 把设备上的文件拉到电脑制定目录   |      |
| 3    | adb shell pm uninstall -k com.packagename     | 卸载某个包                       |      |
| 4    | adb shell pm list packages                    | 查看设备上所有的安装的app        |      |
